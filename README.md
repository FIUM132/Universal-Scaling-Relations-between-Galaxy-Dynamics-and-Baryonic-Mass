摘要
本次研究取用SPARC (Spitzer Photometry & Accurate Rotation Curves) 星系样本的旋转曲线观测数据，对星系动力学参数η（表征内部动力学弥散）与总重子质量M_b之间的标度关系进行了系统性研究。通过对原始172个星系样本进行严格的数据清洗（包括剔除拟合失败点、负拟合优度点及3σ离群点），我们最终获得了一个包含114个高质量星系的干净样本。采用加权最小二乘法（以个体拟合优度R²为权重）并结合Bootstrap重采样进行误差估计，我们得到了稳健的幂律标度关系：η = (2.279 ± 0.046) × M_b^(0.425 ± 0.046)，其线性拟合优度R² = 0.419。残差分析表明模型已充分提取数据中的线性相关性。该标度关系为星系形成与演化模型中重子物质与暗物质相互作用的耦合强度提供了关键的观测约束。
关键词： 星系动力学；标度关系；SPARC；重子质量；旋转曲线；数据清洗

1. 引言
星系物理中的标度关系是连接星系观测属性与内部物理过程（如引力、恒星形成、反馈）的桥梁。其中，动力学参数与质量之间的幂律关系是检验星系形成模型的核心观测依据之一。动力学参数η，定义为星系旋转曲线特征速度的平方，反映了星系整体的运动学弥散状态；而总重子质量M_b包含了恒星与气体等所有可见物质。探索η与M_b之间的普适关系，有助于理解重子物质如何与暗物质晕共同演化。
本研究旨在利用高精度的SPARC数据库，克服以往研究中可能存在的低质量数据污染问题，通过严谨的数据筛选和稳健的统计方法，确立一个精确、可靠的η-M_b标度关系，为理论模型提供更严格的观测约束。
